# Architecture

Playlistour connects music taste signals to real-world concert discovery.

## High-level flow

1. User connects Spotify
2. App reads music taste signals
3. App searches upcoming concerts through Ticketmaster and SeatGeek
4. Events are ranked and explained based on user taste
5. User explores concerts that match their preferences

## Key integrations

- Spotify OAuth and profile/music taste data
- Ticketmaster event search
- SeatGeek event search
- AI-assisted ranking and explanation layer

## Notes

This public repo is a sanitized portfolio version. It does not include API keys, tokens, secrets, user data, or private development notes.
